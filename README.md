# AlphaCare Insurance Solutions - Risk Analytics & Predictive Modeling

## Business Objective
Optimize car insurance marketing in South Africa by identifying low-risk customer segments and building predictive premium models.

## Project Structure
     ├── data/               # Raw and processed data
     ├── notebooks/          # Jupyter EDA notebooks
     ├── src/                # Source code (modular)
     ├── reports/            # Final report + figures
     ├── tests/              # Unit tests
     └── output/             # Plots, models, results

## Key Tasks
- EDA & Loss Ratio Analysis
- A/B Hypothesis Testing (Provinces, Gender, Zipcodes, Margin)
- Predictive Modeling (Premium Prediction)
- Feature Importance & Recommendations
